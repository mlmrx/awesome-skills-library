# Source attribution rules

Awesome Skills Library indexes public skill metadata so maintainers can discover, review, and attribute reusable AI agent skills safely.

## Attribution-first policy

- We preserve original source attribution for every discovered or imported third-party skill.
- We do not claim authorship of third-party skills.
- We do not redistribute third-party skill content unless the license clearly allows redistribution.
- Source-available skills are indexed for discovery and attribution, not copied into this repository's curated `skills/` directory.
- Unknown-license skills are treated as index-only until a maintainer completes manual license review.
- Every imported skill must include the source repository, original author or repository owner, license, and a stable source file URL or commit/file URL when available.
- Manual review is required before moving any third-party skill into the curated `skills/` directory.

## Review expectations

Before copying or adapting third-party skill files, reviewers should confirm:

1. The repository or file license permits redistribution and derivative use.
2. Attribution requirements are captured in catalog metadata.
3. The source URL points to the original repository and file path.
4. Risk tags have been reviewed for commands, credentials, network access, automation, data access, destructive actions, and other sensitive behavior.
5. Any source-available or mixed-license folders remain index-only unless explicit redistribution permission is documented.
