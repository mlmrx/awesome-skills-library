# Vision

Awesome Skills Library exists to make AI agent capabilities reusable, inspectable, and maintainable.

The project treats a skill as infrastructure: a package of instructions, metadata, examples, tests, safety boundaries, and compatibility information. This helps teams share operational know-how without depending on a single agent runtime or private prompt collection.

## Design goals

- Make useful agent workflows easy to discover.
- Make skill behavior reviewable before use.
- Encourage clear input and output contracts.
- Capture safety boundaries and failure modes next to the workflow.
- Keep the initial project lightweight and understandable.

## Non-goals

- This is not a marketplace.
- This is not a hosted agent platform.
- This is not a prompt dump.
- This is not a website or documentation framework.
